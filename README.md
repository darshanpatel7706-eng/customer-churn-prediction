# 🔮 Customer Churn Prediction

A Machine Learning web app that predicts whether a customer will leave a telecom company.

## 🚀 Features
- Predicts customer churn with 77% accuracy
- Handles imbalanced data using SMOTE
- Explains predictions using SHAP waterfall graphs
- Interactive Streamlit dashboard

## 🛠️ Tech Stack
- Python, Pandas, Numpy
- Scikit-learn, Imbalanced-learn
- SHAP, Matplotlib
- Streamlit

## 📊 Model Performance
- Accuracy: 77%
- F1-Score: 0.70
- Algorithm: Random Forest

## ⚙️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Dataset
Telco Customer Churn Dataset from Kaggle
